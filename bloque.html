Elemento SECTION

La etiqueta section se usa para agrupar contenido relacionado dentro de una página.

Elemento ARTICLE

La etiqueta article representa contenido independiente, como una noticia o una entrada de blog.

Elemento ASIDE

La etiqueta aside se usa para contenido relacionado pero no principal, como barras laterales o información extra.

Elemento DIV

El div es un contenedor genérico muy usado para agrupar contenido.
