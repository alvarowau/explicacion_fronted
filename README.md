# Explicación Frontend -- HTML Básico

Este repositorio contiene una pequeña web creada para entender los
conceptos básicos de **HTML**, especialmente la diferencia entre:

-   **Elementos de bloque**
-   **Elementos en línea**

El objetivo es tener un **laboratorio local** donde poder ver cómo se
comportan los distintos elementos HTML.

------------------------------------------------------------------------

# Estructura del proyecto

El proyecto contiene tres páginas HTML conectadas entre sí:

    explicacion_fronted
    │
    ├── index.html
    ├── bloque.html
    └── inline.html

## index.html

Es la página principal del laboratorio.

Desde aquí se puede navegar a los ejemplos de:

-   elementos de bloque
-   elementos en línea

Su objetivo es explicar de forma sencilla qué se va a aprender.

------------------------------------------------------------------------

## bloque.html

Esta página muestra ejemplos de **elementos HTML a nivel de bloque**.

Los elementos de bloque tienen dos características principales:

-   Empiezan siempre en **una nueva línea**
-   Ocupan **todo el ancho disponible**

Ejemplos incluidos en la página:

-   `header`
-   `nav`
-   `section`
-   `article`
-   `aside`
-   `div`
-   `ul`
-   `ol`
-   `table`

------------------------------------------------------------------------

## inline.html

Esta página muestra ejemplos de **elementos HTML en línea**.

Los elementos en línea:

-   **No empiezan una nueva línea**
-   Solo ocupan **el espacio necesario**

Ejemplos incluidos:

-   `span`
-   `strong`
-   `em`
-   `i`
-   `b`
-   `small`
-   `a`
-   `img`
-   `br`
-   `input`
-   `button`

------------------------------------------------------------------------

# Conceptos aprendidos

Durante este proyecto se practican los siguientes conceptos:

## Estructura básica de HTML

``` html
<!DOCTYPE html>
<html>
<head>
<body>
```

## Elementos semánticos

Uso de etiquetas que dan significado al contenido:

-   `header`
-   `nav`
-   `main`
-   `section`
-   `article`
-   `aside`
-   `footer`

## Navegación entre páginas

Uso de enlaces con la etiqueta:

``` html
<a href="pagina.html">
```

para conectar las diferentes páginas del laboratorio.
